Work Chunk Planner User's Guide

Philosophy

A "work chunk" is a work with a single purpose which is planned to be done within a single time frame. The purpose of this system is to help users to plan their work chunk and to assess how the users effectively finished the works.

Format of plan

Project
Specific project that a user is going to work on with this plan.
Work chunk class
General purpose of this plan.
Work chunk type
General methodology used to accomplish this plan.

Evaluating service contribution

Evaluation should be based on how the work chunk was important to move the lab activities forward. For administrative works, the scores are recommended to be 30 or below.
